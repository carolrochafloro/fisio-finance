  <h1 align="center">Fisio Finance</h1>

## Descrição

Gerenciamento de atendimentos e previsão de receita para fisioterapeutas anônimos.

### Tecnologias

- Node.js
- Typescript
- Nest.js

## Instalação

```bash
$ npm install
```

## Executando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
